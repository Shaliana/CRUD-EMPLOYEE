# CRUD-EMPLOYEE
crud with api nodejs and angular ui

# Installation
## Frontend
* install depedencies
```
npm install
```
* run application
```
ng serve --port 4200
```
## Backend (API)
* install depedencies
```
yarn install
```
* set up .env (backend/.env) for PORT and DB_CONNECTION
![image](https://user-images.githubusercontent.com/26450651/227070696-af3161a5-1a17-46e7-9dfb-8936bca15155.png)
* run application
```
yarn dev
```
